In Prometheus

Webhook URL

https://checktheprice.vercel.app/api/prometheus

Value:

ctp_8xL92kQmP7vR4aY2026
The value must exactly match the environment variable you set in Vercel.

This secret is simply a shared password between Prometheus and your website. It lets your website verify that the incoming webhook really came from a trusted source.
